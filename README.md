# Credit Card Fraud Detection using Deep Learning Ensemble

## Overview
This project detects fraudulent credit card transactions using Deep Learning models and ensemble learning techniques.

## Features
- Data Cleaning
- Data Preprocessing
- Feature Engineering
- SMOTE Data Balancing
- MLP Model
- LSTM Model
- GRU Model
- Ensemble Learning
- Fraud Prediction System

## Technologies Used
- Python
- TensorFlow
- Keras
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib

## Models
- Multi Layer Perceptron (MLP)
- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)
- Ensemble Model

## Dataset
Credit Card Fraud Detection Dataset

## Results
The ensemble model achieved strong performance in fraud detection by combining predictions from MLP, LSTM, and GRU models.
